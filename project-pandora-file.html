The Pandora File

A team-made AR escape room experience where the player uses a phone to interact with the main game environment.

UnityC#UDPARCross-Device Interaction

Project Overview

The Pandora File combines my interest in technical implementation with player-facing design.

My main contribution was everything that happened after the networking layer: taking the data sent from the phone and making it behave correctly inside the game world.

One of the biggest takeaways from this project was learning to think about optimization from the beginning instead of treating it as something to fix only at the end.

What I Worked On

  • Worked as the programmer on the gameplay-side logic that turned phone input into actual in-game interaction.
  • Handled the environment interaction layer and the systems that made the received values useful in gameplay.
  • Focused on making the experience respond as smoothly as possible while working within a low-latency requirement.
